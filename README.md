📱 Firebase Phone OTP Verification

📌 About the Project

In this project, I learned how to implement Firebase Phone OTP Verification in an Android application.
This allows users to log in or verify their identity using their mobile number through a one-time password (OTP).

🚀 What I Learned

Creating a Firebase project

Setting up Firebase Authentication

Enabling phone number authentication

Sending and verifying OTP

Redirecting the user after successful verification

🛠️ Technologies Used

Java (Android)

XML

Firebase Authentication

Phone OTP Verification

🔐 Features

User enters phone number

OTP is sent to the mobile number

OTP verification process

Successful login after verification

📖 Conclusion

Through this project, I gained practical experience with Firebase Authentication, especially the Phone OTP verification system, which is widely used in real-world applications.
